## 📜 原文
```
凤凰台上凤凰游，凤去台空江自流。
吴宫花草埋幽径，晋代衣冠成古丘。
三山半落青天外，二水中分白鹭洲。
总为浮云能蔽日，长安不见使人愁。
```
## 🎯 主题
《登金陵凤凰台》主要表达了历史兴衰之感与个人政治情怀。诗中通过凤凰台的荒凉、吴宫花草的埋没、晋代衣冠的消逝，展现了历史变迁的无情与人事的短暂。而壮阔的自然景观与永恒的江流，则与人事变迁形成鲜明对比。最后"浮云蔽日"、"长安不见"的意象，既抒发了诗人对朝廷的思念，也暗含对政治环境的忧虑，体现了李白怀才不遇的苦闷和对理想的执着追求。整首诗以怀古伤今为主线，融合了历史感慨与个人情怀，展现了诗人深沉的历史意识和政治抱负。
## 🏗️ 结构
《登金陵凤凰台》是李白的一首七言律诗，其结构特点体现了精心的章法布局和层次安排。

首先，从整体结构看，这首诗遵循了律诗的典型格式，分为首联、颔联、颈联和尾联四个部分，形成了起承转合的完整结构。

首联"凤凰台上凤凰游，凤去台空江自流"点明题目，通过今昔对比开篇。首句写昔日的繁华景象，次句写今日的萧条冷落，形成了强烈的反差，奠定了全诗的基调。这种对比手法不仅突出了时间流逝带来的变化，也暗示了人事更迭的无奈。

颔联"吴宫花草埋幽径，晋代衣冠成古丘"承接首联的历史感慨，进一步扩展时空范围。诗人选取吴宫和晋代两个历史时期的具体意象，通过"花草埋幽径"和"衣冠成古丘"的描写，深化了时光流逝、物是人非的主题。这一联在内容上与首联形成递进关系，从凤凰台的具体景点扩展到更广阔的历史长河。

颈联"三山半落青天外，二水中分白鹭洲"在结构上是一个转折，由历史感慨转向眼前景色的描绘。诗人视野开阔，描绘了远处的三山和近处的二水分流，展现了宏大的空间感。这一联对仗工整，"三山"对"二水"，"半落青天外"对"中分白鹭洲"，形式上和谐统一，内容上则由虚转实，为尾联的情感抒发作铺垫。

尾联"总为浮云能蔽日，长安不见使人愁"收束全诗，点明诗人的内心情感。这里运用了"浮云蔽日"的典故，暗喻奸臣当道，使得自己无法回到长安，表达了诗人的政治忧思。尾联在情感上达到高潮，将前面的景物描写和历史感慨升华为个人的政治情怀，使全诗在意境上得到升华。

从层次安排看，这首诗呈现出由近及远、由实到虚的特点。首联从眼前的凤凰台写起，颔联扩展到更广阔的历史时空，颈联描绘更远处的自然景观，尾联则转向内心的情感抒发。这种层次安排使诗歌的意境逐渐开阔，情感逐渐深化。

此外，诗中运用了丰富的意象，如"凤凰台"、"江水"、"吴宫"、"晋代衣冠"、"三山"、"二水"、"浮云"、"日"、"长安"等，这些意象既有具体的历史遗迹，又有宏大的自然景观，还有抽象的政治象征，共同构成了一个意蕴丰富的艺术世界。

总的来说，《登金陵凤凰台》在结构上体现了严谨的章法布局和精心的层次安排，通过起承转合的结构、今昔对比的手法、由近及远的视角转换，以及情景交融的艺术处理，使这首诗在形式上工整和谐，在内容上意蕴丰富，在情感上深沉动人，是李白律诗中的代表作之一。
## ✍️ 语言风格
李白的《登金陵凤凰台》在语言风格上展现出独特的艺术魅力，其特点主要体现在以下几个方面：

用词方面，诗人选词精炼而富有表现力。诗中"凤凰"、"台"、"江"、"吴宫"、"花草"、"衣冠"、"三山"、"青天"、"白鹭洲"等意象鲜明具体，构建了丰富的视觉画面。特别是"凤去台空江自流"中的"自流"二字，简洁而意境深远，既描绘了江水自然流淌的景象，又暗含了历史变迁的无奈。同时，诗人巧妙融入"吴宫"、"晋代衣冠"等历史元素，增强了诗歌的历史厚重感，体现了自然景观与人文历史的有机结合。

修辞手法上，这首诗运用了多种技巧。对仗工整是显著特点，如"凤凰台上凤凰游，凤去台空江自流"；"吴宫花草埋幽径，晋代衣冠成古丘"；"三山半落青天外，二水中分白鹭洲"，使诗歌节奏鲜明，音韵和谐。对比手法贯穿全诗，"凤凰游"与"凤去台空"形成盛衰对比，"吴宫花草"与"晋代衣冠"构成历史对比，突显了世事变迁的无情。诗人还善用借景抒情，通过描绘金陵景色，抒发对历史变迁的感慨和对长安的思念。特别是"浮云蔽日"象征奸佞蒙蔽君主，"长安不见"象征诗人远离政治中心、无法施展抱负的苦闷，寓意深远。

句式特点上，全诗采用七言句式，节奏鲜明，气势恢宏。首联写凤凰台的今昔对比，尾联写因浮云蔽日而不见长安的愁绪，形成首尾呼应。诗歌从凤凰台的今昔，到吴晋的历史变迁，再到眼前的自然景观，最后抒发个人情感，层次分明，结构严谨。

音韵方面，全诗平仄相间，音韵和谐，押"游"、"流"、"丘"、"洲"、"愁"等韵，韵脚整齐，增强了诗歌的音乐性。

整体风格上，这首诗体现了李白诗歌豪放中见沉郁的特点。特别是尾联"总为浮云能蔽日，长安不见使人愁"，在豪放的笔调中流露出内心的忧愁。诗人将自然景观、历史遗迹和个人情感融为一体，达到情景交融的艺术效果。诗歌描绘了从凤凰台到三山、白鹭洲的广阔景象，意境开阔，气势磅礴。同时，诗中既有对历史变迁的感慨，又有对现实政治的忧虑，体现了李白诗歌的深度和广度。

《登金陵凤凰台》不仅是一首登高怀古之作，更寄托了诗人对国家政治的关切和个人理想的追求，是李白诗歌中的艺术精品，充分展现了其卓越的语言艺术和深厚的思想内涵。
## 📚 文化札记
《登金陵凤凰台》是李白游览金陵（今南京）时所作的名篇，诗中融入了丰富的历史文化元素。

"凤凰台上凤凰游，凤去台空江自流"中的凤凰台，位于金陵，相传南朝宋文帝元嘉年间，有凤凰飞集于此，因此建台。凤凰在中国传统文化中是祥瑞之鸟，象征吉祥和盛世。这两句既写实景，又暗含对历史变迁的感慨，凤凰已去，台已空，只有长江水依旧奔流不息，表达了物是人非的沧桑感。

"吴宫花草埋幽径，晋代衣冠成古丘"中的吴宫指三国时期东吴的宫殿，金陵曾是东吴都城，孙权在此建都。晋代则指东晋，也曾在金陵（当时称建康）建都。"衣冠"是古代对士大夫、达官贵人的代称。这两句通过对六朝遗迹的描写，表达了朝代更迭、繁华落尽的感慨，昔日吴宫的花草已埋没了小路，晋代的显贵也已成为古墓中的枯骨。

"三山半落青天外，二水中分白鹭洲"描绘了金陵的地理景观。三山指南京西南长江岸边的三座山，即三山矶。白鹭洲是长江中的一个沙洲，因常有白鹭栖息而得名。这两句生动地描绘了金陵的山水之美，三座山峰若隐若现，长江被白鹭洲分为两支水流。

"总为浮云能蔽日，长安不见使人愁"中的"浮云蔽日"是一个典故，出自《论语》和陆贾《新语》，比喻小人蒙蔽君主，使贤者不得进用。长安是唐朝的都城，这里代指朝廷。这两句表达了诗人对朝廷政治的忧虑，担心小人当道，蒙蔽君主，自己无法回到朝廷施展抱负，因此感到忧愁。

整首诗既有对六朝兴衰的历史感慨，又有对现实政治的忧虑，体现了李白作为一位有政治抱负的诗人的忧国忧民情怀。
## 🌅 创作背景
《登金陵凤凰台》创作于唐玄宗天宝年间（约742-756年），地点在金陵（今南京）。当时李白正处于人生的重要转折期，他曾短暂入京供奉翰林，但因性格傲岸不羁，不愿逢迎权贵而离开长安，开始了漫游生活。这一时期，李白仕途受挫，政治理想无法实现，内心充满忧愤与失落。

历史背景方面，天宝年间是唐朝由盛转衰的关键时期。表面繁荣下，朝廷内部矛盾重重，杨贵妃专宠，杨国忠把持朝政，政治日益腐败，安史之乱的阴影已经显现。李白作为有政治抱负的知识分子，对国家命运深感忧虑。

金陵作为六朝古都，承载着丰富的历史记忆。李白登临凤凰台，面对"凤去台空"的景象，联想到六朝兴衰，触景生情。诗中"吴宫花草埋幽径，晋代衣冠成古丘"描绘了历史遗迹的荒凉，引发对历史变迁的感慨。"三山半落青天外，二水中分白鹭洲"则展现了金陵壮丽的自然景观。最后两句"总为浮云能蔽日，长安不见使人愁"，既表达了对朝廷政治黑暗的忧虑，也暗含了自己政治理想无法实现的苦闷，以及对长安的思念之情。这首诗融合了历史感慨、自然景观和个人情怀，体现了李白诗歌的豪放风格和深刻思想。
## 💭 情感基调
《登金陵凤凰台》的情感基调主要表现为怀古伤今、忧思惆怅。诗人通过凤凰台的今昔对比，抒发了对历史兴衰的感慨，流露出物是人非的沧桑感。尾联"长安不见使人愁"更直接表达了政治忧思和思乡之情。

氛围特点则呈现出沉郁苍凉与壮阔开阔的交织。前两联营造出历史沧桑的沉郁氛围，而颈联描绘的自然景观则展现出壮阔之美，形成鲜明对比。整首诗在怀古与现实、个人与历史的交织中，形成了一种既苍凉又壮美的独特氛围。
## 🔍 赏析
《登金陵凤凰台》是李白七言律诗的代表作，通过登高望远的视角，构建了时空交错的宏大意境。首联以凤凰台传说起兴，"凤凰游"与"台空"形成鲜明对比，暗示繁华已逝、物是人非；颔联跨越数百年历史，从三国吴国到东晋，展现历史无情更迭；颈联转向空间描写，"三山半落青天外，二水中分白鹭洲"以壮阔的自然景象形成开阔深远的视觉效果；尾联将自然景象与政治现实巧妙结合，"浮云蔽日"象征奸佞当道，"长安不见"表达政治忧虑。全诗通过虚实结合、时空交错、象征暗示等手法，营造出既壮阔又深沉、既怀古又伤今的复杂意境。

语言上，这首诗体现了李白豪放中见深沉的风格，语言简洁凝练，意象鲜明。对比手法贯穿全诗，如"凤凰游"与"台空"、"吴宫花草"与"埋幽径"等，强化了诗歌的张力。象征手法丰富，凤凰象征盛世祥瑞，"凤去台空"象征盛世不再；"浮云蔽日"象征奸佞当道。诗歌还巧妙运用了凤凰台传说、吴宫和晋代衣冠等历史典故，增强了历史厚重感。音韵方面，格律严谨，平仄协调，押平声韵，读来朗朗上口。

情感表达上，诗歌流露出诗人对历史沧桑的感慨，对自然美景的赞美，以及对政治前途的忧虑。通过历史遗迹与自然景观的对照，探讨了历史与现实的辩证关系、个人命运与时代变迁的联系、自然永恒与人事短暂的对比等深刻主题。诗人在登高望远中，既看到历史的沧桑，又感受到现实的无奈，既有对自然美景的赞美，又有对个人命运的忧虑，体现了李白"出世"与"入世"思想的矛盾统一。

文学价值方面，这首诗将怀古、写景、抒情完美结合，代表了唐代怀古咏史诗的典型风格，体现了李白诗歌的艺术高度和思想深度。它所体现的登高望远、怀古伤今的创作模式，以及将个人情感与历史思考相结合的艺术手法，对后世文学产生了深远影响，成为中国古典诗歌的重要传统。同时，诗中提到的金陵多处历史遗迹，也为研究南京历史文化提供了珍贵的文学资料，具有重要的历史文化价值。
