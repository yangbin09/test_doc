## 📜 原文
```
轮台城头夜吹角，轮台城北旄头落。
羽书昨夜过渠黎，单于已在金山西。
戍楼西望烟尘黑，汉兵屯在轮台北。
上将拥旄西出征，平明吹笛大军行。
四边伐鼓雪海涌，三军大呼阴山动。
虏塞兵气连云屯，战场白骨缠草根。
剑河风急雪片阔，沙口石冻马蹄脱。
亚相勤王甘苦辛，誓将报主静边尘。
古来青史谁不见，今见功名胜古人。
```
## 🎯 主题
《轮台歌奉送封大夫出师西征》主要主题思想是歌颂唐军将士不畏艰险、英勇出征的爱国精神。诗中通过描绘轮台城头军号响起、紧急军情传来、封常青统帅大军西征等场景，展现了边塞战争的紧张氛围和唐军出征的壮观场面。诗人以"四边伐鼓雪海涌，三军大呼阴山动"等生动描绘，表现了唐军气势磅礴的威武形象；同时通过"剑河风急雪片阔，沙口石冻马蹄脱"等诗句，反映了西域环境的恶劣和将士们所面临的艰苦考验。全诗赞扬了封常青"勤王甘苦辛，誓将报主静边尘"的忠勇精神，并预言其功绩将超越古人，名垂青史，体现了诗人强烈的爱国情怀和对英雄的崇敬。
## 🏗️ 结构
《轮台歌奉送封大夫出师西征》是岑参边塞诗的代表作之一，其结构特点鲜明，章法布局严谨，层次安排有序。

从整体章法看，这首七言古诗共十四句，可分为四个有机联系的部分，形成"战前形势—出征场景—战场描写—功勋颂扬"的完整结构。这种布局使诗歌层次分明，逻辑清晰，既展现了战争的壮阔场面，又突出了封大夫的英雄形象。

第一部分为战前形势描写，包括前四句。开篇"轮台城头夜吹角，轮台城北旄头落"点明地点和紧张气氛，"羽书昨夜过渠黎，单于已在金山西"进一步说明敌情紧急，"戍楼西望烟尘黑，汉兵屯在轮台北"则勾勒出对峙局面。这部分以"轮台"为中心，通过空间转换和时间推移，营造出战争一触即发的紧张氛围。

第二部分为出征场景描写，包括第五至八句。"上将拥旄西出征，平明吹笛大军行"描写封大夫率军出征的威武场面，"四边伐鼓雪海涌，三军大呼阴山动"则用夸张手法渲染军威之盛。这部分从夜晚到天明的时间推移，展现了唐军出征的磅礴气势。

第三部分为战场环境描写，包括第九至十二句。"虏塞兵气连云屯，战场白骨缠草根"描绘敌军强大和战争残酷，"剑河风急雪片阔，沙口石冻马蹄脱"则具体描写边塞战场的恶劣环境。这部分通过对比手法，既写敌军强大，又写环境艰苦，突显征战的艰难。

第四部分为功勋颂扬，包括最后两句。"亚相勤王甘苦辛，誓将报主静边尘"赞扬封大夫忠心报国，"古来青史谁不见，今见功名胜古人"则高度评价其功绩超越古人。这部分总结全诗，点明主题，使结构完整。

在层次安排上，诗歌采用时间与空间双线并进的方式。时间上从夜晚到天明再到征战过程，空间上从轮台城到金山再到战场（剑河、沙口），时空转换自然流畅。情感层次也呈现递进关系：从紧张到激昂，再到悲壮，最后是崇敬，情感变化丰富而有层次。

此外，诗歌结构还体现了首尾呼应的艺术特色。开篇写战事起，结尾写功成；开头写敌军压境，结尾写功胜古人，形成完整闭环。对比手法的运用也增强了结构张力，如敌军兵强马壮与战场白骨的对比，艰苦环境与将士决心的对比，古人与今人的对比等。

总体而言，这首诗结构严谨而灵活，层次分明而连贯，既有宏观的章法布局，又有微观的细节描写，充分展现了岑参边塞诗的艺术特色和唐代边塞诗的典型结构模式。
## ✍️ 语言风格
岑参的《轮台歌奉送封大夫出师西征》是唐代边塞诗的代表作，其语言风格特点鲜明，主要表现在以下几个方面：

在用词方面，诗中大量运用边塞特色词汇，如"轮台"、"渠黎"、"金山"、"阴山"、"剑河"、"沙口"等边塞地名，营造出浓郁的边塞地域氛围。军事术语如"吹角"、"羽书"、"戍楼"、"上将"、"拥旄"、"伐鼓"、"三军"等，增强了诗歌的军事色彩。同时，诗人选用了一系列鲜明有力的意象，如"旄头"、"烟尘黑"、"雪海涌"、"白骨缠草根"、"风急雪片阔"等，通过具体可感的意象生动描绘边塞战争场景。这些词汇的运用既真实反映了边塞军旅生活，又形成了强烈的视觉冲击力。

在修辞手法上，诗人灵活运用多种技巧增强艺术表现力。夸张手法如"四边伐鼓雪海涌，三军大呼阴山动"，通过夸大军威气势，展现出震撼人心的力量。对偶手法如"轮台城头夜吹角，轮台城北旄头落"、"剑河风急雪片阔，沙口石冻马蹄脱"等，形成整齐对称的结构美。比喻修辞如"雪海涌"将鼓声比作汹涌的海浪，形象生动。象征手法如"旄头落"象征战争胜利，"白骨缠草根"象征战争的残酷。此外，诗人还善于运用衬托手法，通过对恶劣自然环境的描写("风急雪片阔，石冻马蹄脱")，衬托出征将士的英勇无畏。

诗歌整体风格雄浑豪放，气势磅礴，如"四边伐鼓雪海涌，三军大呼阴山动"等句，展现出强大的艺术感染力。诗人以亲身经历为基础，真实地描绘了边塞战争的场景和将士们的英勇形象，使诗歌具有强烈的真实感。情感表达上，诗中既有对战争残酷的揭示("战场白骨缠草根")，又有对将士英勇的赞美("亚相勤王甘苦辛")，还有对胜利的信心("今见功名胜古人")，情感层次丰富而复杂。

在音韵方面，全诗以七言为主，节奏整齐有力，适合表现军旅题材。押韵工整，全诗押"落、西、北、行、动、根、脱、尘、人"等韵，音韵和谐，增强了诗歌的音乐性和节奏感。

结构上，全诗十二句，层次分明，从战前准备到出征场面，再到环境描写和人物赞美，最后以对胜利的展望作结，结构完整而严谨，展现出诗人高超的艺术构思能力。

总体而言，岑参的这首诗通过丰富的军事词汇、生动的意象描绘和多样的修辞手法，展现了唐代边塞诗的雄浑豪放风格，同时也表达了诗人对战争的复杂情感和对将士们的赞美之情，是唐代边塞诗的经典之作。
## 📚 文化札记
《轮台歌奉送封大夫出师西征》是岑参的代表作之一，属于唐代边塞诗的典范。以下是诗中重要的文化背景注释：

轮台：地名，位于今新疆维吾尔自治区乌鲁木齐市西北，是唐代西北边疆的重要军事据点，也是安西都护府辖下的战略要地。汉代张骞出使西域后，轮台成为丝绸之路上的重要站点，唐代时更是军事防线的核心。

旄头：指星宿名，即二十八宿中的昴宿。古人认为旄头星坠落是战争胜利的吉兆，诗中"旄头落"暗示唐军必将取胜，是一种占星术的体现。

羽书：古代紧急军情文书，因插有羽毛而得名，表示十万火急，需要迅速传递。在唐代军事通讯体系中，羽书是最紧急的军情传递方式，通常用于战情紧急时。

渠黎：古代西域国家或地区名，在今新疆境内，是唐代丝绸之路上的一个站点，也是军事要地。

单于：本是匈奴君主的称号，后来在唐代诗歌中泛指北方游牧民族（如突厥、回纥等）的首领。这是唐代诗歌中常见的借代手法。

金山：即阿尔泰山，位于今新疆北部和蒙古人民共和国西部，是唐代西北边疆的重要自然屏障，也是与北方游牧民族争夺的战略要地。

汉兵：唐代诗歌中常以"汉"代指"唐"，是一种常见的文学修辞手法。诗中"汉兵"实指唐朝军队。

上将拥旄：指封大夫（封常清）持旌节出征。拥旄是古代高级将领出征时持旄节作为权力象征的仪式，旄节是用牦牛尾装饰的旗帜，是皇帝授予将领的权力象征。

雪海：形容广阔的雪原，可能指今新疆北部或中亚地区的雪原，反映了唐代西域地区严寒的自然环境。

阴山：山脉名，在今内蒙古自治区中部，是古代北方的重要地理标志，也是中原王朝与北方游牧民族的天然分界线。

虏塞：指敌方的边塞要地。"虏"是古代中原对北方游牧民族的称呼，带有贬义色彩。

剑河、沙口：唐代西域地区的地名，具体位置已难确考，但从诗中描述可知是气候恶劣、地形复杂的边塞地区。

亚相：指封大夫，即封常清，唐朝著名将领。天宝年间（742-756年），封常清曾任安西节度使，负责西域地区的军事防务。"亚相"是对其官职的尊称。

勤王：为君主效力，保卫国家。这是古代臣子的政治伦理和道德准则，体现了忠君爱国的思想。

青史：史书，因古代用竹简记事，竹简呈青色，故称史书为青史。"青史留名"是指在历史上留下美名，是中国传统价值观中对人生成就的最高追求之一。

岑参（约715-770）：唐代著名诗人，与高适并称为"高岑"，是盛唐边塞诗的代表人物。他曾两次出塞，亲历边塞生活，其边塞诗风格雄奇豪放，气势磅礴，真实反映了唐代边疆战争和边塞生活。

这首诗创作于天宝年间，当时唐朝与吐蕃、大食（阿拉伯帝国）等在西域的争夺日益激烈，边疆战事频繁。封常清出师西征正是在这样的历史背景下进行的，诗中既表现了战争的残酷，也表达了对将领功绩的赞美和对胜利的期盼。
## 🌅 创作背景
岑参的《轮台歌奉送封大夫出师西征》创作于唐玄宗天宝末年（约755年前后），地点在唐代西域的轮台（今新疆维吾尔自治区境内）。当时岑参正在封常清幕府中任职，这是他第二次出塞时期。

岑参此时年届不惑，正处于人生的壮年阶段。他曾在安西节度使高仙芝幕府中任职，后来转至北庭都护封常清的幕府。长期生活在边塞的岑参，对西域的自然环境和军旅生活有着深刻的体验和认识。他一方面怀有建功立业的抱负，另一方面也深知边塞生活的艰辛和战争的残酷。这种复杂的心境在诗中得到了充分体现，既有对出征将士的赞美，也有对战争残酷性的真实描绘。

这首诗创作的历史背景是唐朝由盛转衰的关键时期。天宝年间，虽然表面仍维持着开元盛世的繁荣，但各种社会矛盾已经开始显现。唐朝在西域设立了安西都护府和北庭都护府，以加强对西域的控制，但同时也面临着来自吐蕃、突厥等势力的挑战。封常清出师西征，正是为了应对西域地区的军事威胁。诗中"单于已在金山西"一句，表明当时有北方游牧民族的活动威胁到了唐朝的边疆安全。此外，天宝十四年（755年）安史之乱爆发，唐朝由盛转衰，虽然这首诗可能创作于安史之乱爆发前，但当时唐朝的内部矛盾已经相当严重。

岑参创作这首诗的直接动机是奉送封常清出师西征。作为封常清幕府中的一员，岑参有责任和义务为上司出征送行，并以诗歌的形式表达对这次军事行动的支持和对将士们的赞美。其灵感来源主要来自几个方面：一是他长期生活在边塞，亲身体验了边塞的自然环境和军旅生活；二是封常清出师西征这一军事行动本身，诗中描绘的军队出征的壮观场面，很可能就是作者亲眼所见；三是西域地区独特的自然风光，如"雪海"、"阴山"、"剑河"等，为诗歌提供了丰富的意象；四是中国自古以来以诗歌记录军事行动、赞美将领功绩的传统，岑参的这首诗也继承了这一传统。
## 💭 情感基调
这首《轮台歌奉送封大夫出师西征》的情感基调可概括为悲壮激昂、雄浑豪迈。诗中通过"夜吹角"、"羽书过"、"伐鼓"、"大呼"等战争声响的描写，营造出紧张激烈的氛围；"剑河风急雪片阔，沙口石冻马蹄脱"勾勒出严酷苍凉的边塞环境；"四边伐鼓雪海涌，三军大呼阴山动"则展现了壮阔雄浑的战争场面。同时，"战场白骨缠草根"流露出对战争残酷的悲悯，而"亚相勤王甘苦辛，誓将报主静边尘"和"今见功名胜古人"又表达了将士们报国立功的豪情壮志。整首诗在描绘战争残酷与边塞艰苦的同时，也歌颂了将士们的英勇无畏和爱国情怀，形成了悲壮与豪迈交织的独特艺术风格。
## 🔍 赏析
《轮台歌奉送封大夫出师西征》是岑参边塞诗的代表作，通过描绘唐军出征西域的壮阔场景，展现了边塞诗的独特魅力。在意境营造上，诗人采用时空交错的叙事方式，从"轮台城头夜吹角"到"平明吹笛大军行"，形成时间上的连贯；空间上从轮台到金山、雪海、阴山，构建了广阔的地理视野。诗人善用对比手法，如"戍楼西望烟尘黑，汉兵屯在轮台北"形成敌我对峙的紧张感；"四边伐鼓雪海涌，三军大呼阴山动"通过动静结合，表现唐军威武气势。细节描写如"剑河风急雪片阔，沙口石冻马蹄脱"增强了真实感，使读者仿佛置身于边塞战场。

语言特色方面，诗歌采用七言句式，简洁有力，节奏感强。对偶和排比手法贯穿全诗，如"羽书昨夜过渠黎，单于已在金山西"等对偶句使结构严谨，节奏鲜明。诗人善用比喻和象征，"旄头落"象征敌人覆灭，"烟尘黑"暗示战争残酷。夸张手法如"四边伐鼓雪海涌，三军大呼阴山动"极大增强了诗歌气势。边塞特有的地理名词和军事术语的运用，不仅增加了真实感，也体现了诗人对边塞生活的熟悉。

情感表达上，诗人既表达了对国家安定的关切和对战争胜利的期望，如"亚相勤王甘苦辛，誓将报主静边尘"；也表现了对战争残酷性的认识和对士兵的同情，如"战场白骨缠草根"。同时，诗人赞美了唐军英勇无畏的精神，体现了爱国主义情怀和建功立业的思想，"古来青史谁不见，今见功名胜古人"直接表达了建功立业、名垂青史的思想。

作为边塞诗的典范之作，这首诗具有重要的文学价值和深远影响。它集中体现了边塞诗的特点，在艺术表现上成就卓著，对后世边塞诗创作产生深远影响。诗中的地理名词和军事术语为研究唐代边塞地理和军事制度提供了宝贵资料，所体现的爱国主义情怀和建功立业思想，激励着一代又一代中国人。同时，诗中对战争残酷性的揭示，也使人们对和平有了更深刻的认识和珍视。
